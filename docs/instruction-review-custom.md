# ファイル別レビュー観点

ファイル名に対応する観点をもとにレビューを実施すること。

## Java

### ENSYU2_1.java
- Date型、SimpleDateFormat型、Calendar型を使用していること
- Java8以降のTime APIを使用していないこと
- コンソールなどに現在から100日後の日付を出力していること
- Java8以降のTime APIの使用が禁止されていること

### ENSYU2_2.java
- LocalDate型とDateTimeFormatterを使用していること
- Java8以降のTime APIを使用していること
- コンソールなどに現在から100日後の日付を出力していること

### ENSYU3_1.java, ENSYU3_1.txt
- Set型の変数を使ってsetの回答をしていること
- List型の変数に5人の生徒のテスト点数を格納していること
- Map型の変数に過去の総理大臣の名前と任期（順序は不問）を格納していること

### ENSYU3_2.java
- Heroインスタンスを2つ生成していること
- HeroインスタンスをArrayListに格納し、for文で名前を出力していること

### ENSYU3_3.java
- Heroインスタンスを2つ生成していること
- HeroインスタンスをArrayListに格納し、for文で名前を出力していること
